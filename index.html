Installation

Portal: Not a Lie nécessite p2ce pour fonctionner.

  1. Téléchargez et installez p2ce sur Steam.
  2. Téléchargez la dernière version de Not a Lie depuis GitHub.
  3. Extrayez le dossier dans votre répertoire steamapps/sourcemod
  4. Redémarrez Steam puis Portal Not a Lie va apparaître.

Accédez au code source et contribuez au projet sur GitHub (après avoir obtenu Portal 2).
